# codacy-coffeescript

Offers coffeescript support for Codacy

# Developing

## Dependencies
There are some dependencies not installed by the bootstrap script. You will need
* JDK
* Scala
* sbt
* node
* npm

## Bootstrapping
Bootstrap the development environment with

```bash
sh script/bootstrap
```

## Testing
Test the project with

```bash
sh script/bootstrap
```
